Neste diretório estão as classes:
Super classe - Personagem :
    1. nome; 
    2. classe;
    3. nivel;
    4. pontos de vida;
    5. poder base;
    6. habilidade;
E as subclasses:
Mago:
    Herda os atributos de Personagem e inclui o seu próprio atributo "inteligencia"
    Habilidade: Usuario digita a habilidade
Guerreiro:
    Herda os atributos de Personagem e inclui o seu próprio atributo "força"
    Habilidade: Usuario digita a habilidade 
Assassino:
    Herda os atributos de Personagem e inclui o seu próprio atributo "furtividade"
    Habilidade: Usuario digita a habilidade
